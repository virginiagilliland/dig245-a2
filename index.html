Opening a Candy Bar: How to View the HTML Source of a Webpage in Chrome:

  1. Go to your favorite website, mine is currently Hiking Project. Imagine the part of the website you can see as the wrapper, and we want to get to the candy itself - the source code.
  2. Next, right click anywhere on the webpage
  3. Select "inspect" from the dropbdown box that appears. This will allow you to access the source code - the candy bar itself!
  4. A new panel with the HTML source code will appear
  5. If you cannot view the code, make sure "Elements" is selected from the list of options across the top of the panel
